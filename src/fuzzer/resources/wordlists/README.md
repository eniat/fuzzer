# Wordlists

Place local wordlists in this directory to use short-name CLI references.

Wordlists are not included in this repository.

Example:

- `sql.txt` can be referenced as `--swordlist sql`
- `XSS-Jhaddix.txt` can be referenced as `--xwordlist XSS-Jhaddix`
- `LFI-Jhaddix.txt` can be referenced as `--pwordlist LFI-Jhaddix`

Recommended source:

- https://github.com/danielmiessler/SecLists

Do not commit third-party wordlists or sensitive testing data.
